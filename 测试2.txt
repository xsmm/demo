命令行指令：
cd         进入文件夹
Dir         显示当前目录下的文件
Md        创建文件夹
Copy nul    创建文件
Cd..        返回上层目录
Cd/        返回根目录
Ctrl+c双击  停止运行


Sass命令执行
Sass sass/index.scss css/index.css
返回当前目录
Sass --watch sass/index.scss:css/index.css(监听文件)
Sass --watch

Git指令

Git reset --hard HEAD-1 回滚上一个版本
Git clone 服务器仓库地址（一次）

从本地推送到服务器的操作流程：
1.Git add .
2.Git commit -m ‘提交信息’
  Git pull 将服务器的版本拉去下来到本地
3.Git push 推送}
